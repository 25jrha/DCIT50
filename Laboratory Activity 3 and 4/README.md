## Lab 3 & 4: Coffee Shop Rewards System

**Objectives:**
- Implement default and parameterized constructors.
- Use static variables and methods.
- Apply OOP principles in a real-world context.

**Functionality:**
- Track customer points and allow reward redemption.
- Create multiple customer objects.
- Display customer info and total registered customers.
- Calculate redeemable rewards based on points.

**Key Concepts:**
- Constructors
- Static variables and methods
- Conditional logic and state management

**Files:**
- `Customer.java`
- `CoffeeShop.java`
