## Lab 1: Library Book Management System

**Objectives:**
- Implement arrays to store book data.
- Use loops, conditionals, and methods for program logic.
- Develop a menu-driven application for basic library management.

**Functionality:**
- Display a main menu with options: Add Book, Update Book Status, Show All Books, Generate Report, Exit.
- Add new books with default status "Available" (limit of 5 books).
- Update book status between "Available" and "Borrowed".
- Show a list of all books with their current status.
- Generate a report showing total books, available books, and borrowed books.
- Handle invalid menu selections gracefully.

**Key Concepts:**
- Arrays
- Loops and conditional statements
- Methods and modular programming

**File:**
- `Lab1_JavaReview.java`
