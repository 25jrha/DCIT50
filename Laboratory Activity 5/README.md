## Lab 5: School Management System

**Objectives:**
- Implement inheritance using the `extends` keyword.
- Differentiate between IS-A and HAS-A relationships.
- Apply aggregation for associated classes.

- Model `Person`, `Student`, and `Teacher` classes.
- Teachers can have multiple courses.
- Display detailed information for students, teachers, and courses.

**Key Concepts:**
- Inheritance
- Aggregation
- Encapsulation and polymorphism
