## Finals Lab Examination

**Objectives:**
- Use Java Swing to create a GUI task management system.
- Apply OOP concepts: abstraction, encapsulation, inheritance, and polymorphism.

**Functionality:**
- Create and manage tasks using `Task` and `TimedTask` classes.
- Display tasks in a JTable.
- Add new tasks using a separate form window.
- Validate user input and update the table dynamically.

**Key Concepts:**
- GUI Development (Java Swing)
- Abstraction and encapsulation
- Event-driven programming

**Files:**
- `model/AbstractTask.java`
- `model/Task.java`
- `model/TimedTask.java`
- `model/Status.java`
- `controller/TaskManager.java`
- `ui/MainApplicationWindow.java`
- `ui/TaskForm.java`
