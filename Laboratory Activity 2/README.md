## Lab 2: Working with Java classes and objectst

**Objectives:**
- Define and implement classes in Java.
- Instantiate objects and use constructors, methods, and properties.
- Apply basic OOP concepts in a small project.

**Functionality:**
- Create a `Book` class with attributes: title, author, year published, current page.
- Implement methods to display details, change pages, and update authors.
- Store books in an array and filter books based on publication year.

**Key Concepts:**
- Classes and objects
- Constructors and methods
- Arrays of objects

**Files:**
- `Book.java`
- `MyBookList.java`
