## Lab 6: Smart Home Control System

**Objectives:**
- Implement method overriding for different device behaviors.
- Use polymorphic references and dynamic binding.
- Manage device states and control multiple devices.

**Functionality:**
- Control devices: Air Conditioner, Lamp Shade, Television, Microwave Oven.
- Adjust device attributes (brightness, temperature, volume, etc.).
- Turn devices on/off and check current device status.
- Count powered-on devices in a home.

**Key Concepts:**
- Polymorphism
- Inheritance
- Method overriding and encapsulation

**Files:**
- `SmartHomeMain.java`
